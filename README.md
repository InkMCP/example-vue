# Vue

Learn how to deploy a Vue.js app with Ink MCP using a simple prompt.

## Tutorial

Full step-by-step tutorial: [docs.ml.ink/examples/frontend/vue](https://docs.ml.ink/examples/frontend/vue)

### Prerequisites

- An [Ink](https://ml.ink) account
- Ink MCP connected to your AI agent ([setup guide](https://docs.ml.ink/quick-start))

### Quick Deploy

1. Connect Ink MCP to your agent
2. Prompt: "Create a Vue 3 app with a todo list that persists to localStorage. Use the Composition API and Tailwind CSS."
3. Prompt: "Deploy with Ink"
