# Example Vue App

A simple Vue + Vite example app, ready to deploy.

## Setup

```bash
npm install
npm run dev
```

## Deploy with Ink

Connect the [Ink MCP server](https://docs.ml.ink/quick-start) to your AI agent and prompt:

> Deploy this Vue app with Ink.
